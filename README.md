# Sacred Texts — site

Landing page and privacy policy for the **Sacred Texts** mobile app
(multi-faith scripture reader by BannedProduct Media Inc.).

## Enable GitHub Pages (one time)

Settings → **Pages** → Build and deployment → Source: **Deploy from a branch**
→ Branch: **main** / **/(root)** → **Save**.

After ~1 minute the site is live:

- Home:    https://rceasar01.github.io/sacred-texts-app/
- Privacy: https://rceasar01.github.io/sacred-texts-app/privacy.html

Use the privacy URL for the App Store and Google Play submissions.
